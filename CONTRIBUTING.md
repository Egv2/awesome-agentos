# Contributing to Awesome AgentOS

Thank you for your interest in contributing to Awesome AgentOS! This list is a curated collection of the best tools, frameworks, and platforms for building agentic operating systems.

## Guidelines

Please ensure your contribution meets these criteria before submitting a pull request:

- **Quality over quantity.** Only include projects that are genuinely useful and well-maintained.
- **Relevance.** The project must relate to agentic operating systems — agent frameworks, computer-use automation, web agents, voice AI, visual AI, developer tools, LLM infrastructure, security, data/knowledge systems, productivity tools, or MCP integrations.
- **Active maintenance.** The project should be actively maintained. Archived, deprecated, or unmaintained projects are not accepted.
- **Open source.** The project must be open source.
- **Proper formatting.** Follow the existing format:
  ```
  - [Project Name](https://github.com/author/project) - Description starting with an uppercase letter and ending with a period.
  ```
- **Objective descriptions.** Write neutral, factual descriptions. Avoid marketing language, superlatives, and promotional tone.

## How to Contribute

1. **Search the list** to ensure the project is not already included.
2. **Check the project quality:** It should be open-source, actively maintained, and relevant to the AgentOS theme.
3. **Create a Pull Request:**
   - Add the project to the most appropriate category.
   - Use the format: `- [Project Name](URL) - Description.`
   - Keep descriptions objective, concise, and in English.
4. **Wait for review.** Pull requests will be reviewed as soon as possible.

## What Makes a Good Description

- Starts with an uppercase letter.
- Ends with a period.
- Describes what the project *does*, not what it *is* (e.g., "Translates SRT files using AI" not "An AI translation tool").
- Avoids superlatives like "best", "leading", "state-of-the-art" unless they are part of the project's official name.
- Does not include the project name in the description.

Thank you for helping make this list awesome!
